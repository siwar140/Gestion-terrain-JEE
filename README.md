Gestion Terrain - JEE Application
Overview
Gestion Terrain is a comprehensive web application built with Java Enterprise Edition (JEE) for managing sports field reservations, client management, and payment tracking. This application provides a complete solution for sports facility administrators to efficiently manage terrains, clients, reservations, and associated payments.

Features
Core Functionality
Client Management: Add, view, and manage client information

Terrain Management: Create, update, and manage sports fields

Reservation System: Book terrains with date and time management

Payment Processing: Track and manage payments for reservations

Advanced Features
Dashboard: Centralized view of system statistics and recent activities

Football Clients: Specialized view for football-related client management

Reservation Filters:

Reservations by terrain

March 2025 reservations view

Unreserved terrains identification

Payment Analysis: Total amount per client with detailed breakdown

Reservation Management: Delete and modify existing reservations

Payment Modification: Update payment records as needed

Technology Stack
Backend
Java EE (JEE)

Servlets (MainServlet.java)

JPA/Hibernate for ORM

EJB (Enterprise JavaBeans) for business logic

Frontend
JSP (JavaServer Pages)

HTML/CSS

Bootstrap (assumed for responsive design)

Database
MySQL (via Terrain-ds.xml datasource configuration)

JPA for database operations
